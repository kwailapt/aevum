The architecture of this system is built upon the 80/20 compounding leverage principle:
1. **Natural Language Agent Harness (NLAH)**: Abolishes hardcoded Python control flows. All evolutionary boundaries, constraints, and role switches are written into a Markdown contract that can be tracked and self-modified via Git.

3. **Mandatory Context Resets**: Flushes the LLM memory entirely after every single Tick. State is passed ONLY through a "Structured Handoff Document", completely eliminating attention decay (Lost in the middle).
4. **Zero-Sum Self-Play Engine (AlphaZero Mapping)**: Eradicates human priors. The Evaluator automatically generates the most rigorous mathematical/compilation constraints possible; the Generator attempts to write the minimal code required to pass them.
5. **Topological Energy Minimization (AlphaFold Mapping)**: Introduces minimal discrete matrices (e.g., I Ching / BioGeometry) as rigid structural constraints. Mutated code must conform to topological symmetry; violators are deemed "high-energy collisions" and are annihilated in $O(1)$ time.
